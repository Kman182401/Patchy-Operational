# Vault Setup — Manual Steps

Complete these three steps after opening the vault in Obsidian:

1. **Install community plugins:**
   - Go to Settings → Community Plugins → Turn on community plugins
   - Click "Browse" and search for each of these, then click Install and Enable:
     - **Dataview** (by Michael Brenan) — powers automatic task lists and changelog rollups
     - **Templater** (by SilentVoid) — enables the note templates in `_templates/`
     - **Bases** (search for current name) — adds database-style views for your notes

2. **Restart Obsidian** — close and reopen so all plugin settings take effect.

3. **Verify everything works:**
   - Open the Dashboard — you should see task tables and changelog entries
   - Open the Work Board — you should see your open todos and upgrades
   - Try creating a new note from a template (click the Templater icon or use the command palette)

**This note can be deleted after setup is complete.**
